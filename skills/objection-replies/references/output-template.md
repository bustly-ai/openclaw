# Output Template

Use this template when writing the final response for **Comment Objection Replies**.

## 1. Conclusion / Verdict
- One-line answer
- Confidence level

## 2. Evidence / Context
- Facts observed
- Missing data
- Key constraint

## 3. Preflight and Execution
- Auth / access status
- Command or request shape
- Expected success signal

## 4. Result / Error Log
- What succeeded
- What failed or was not verified
- Retry / rollback note

## 5. Validation Loop
- Recheck: the targeted complaint or refund theme declines
- Review window
- Improvement / no-change / failure definition
