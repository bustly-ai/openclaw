# Output Template

Use this template when writing the final response for **Cross Platform Recut Planner**.

## 1. Conclusion / Verdict
- One-line answer
- Confidence level

## 2. Evidence / Context
- Facts observed
- Missing data
- Key constraint

## 3. Priority Actions
- P0
- P1
- P2

## 4. Risks / Caveats
- Main risk
- What could change the recommendation

## 5. Validation Loop
- Recheck: the draft is immediately reusable by the next owner
- Review window
- Improvement / no-change / failure definition
